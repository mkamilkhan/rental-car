# Admin Dashboard - Offroad Rental Hub

Separate admin dashboard application for managing bookings and vehicles.

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Runs on http://localhost:3001 (or next available port)

## Build for Production

```bash
npm run build
```

## Environment Variables

Create a `.env` file in the root directory:

```
REACT_APP_API_URL=http://localhost:5000
```

## Features

- Admin login
- View all bookings
- Manage vehicles (add, edit, delete)
- View statistics and analytics

## Deployment

This can be deployed separately from the main website on a different domain/subdomain.
